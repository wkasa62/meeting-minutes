세금계산서 상세 — 일괄/개별 변경 기능 (신규)

추가된 기능
  • 상세 화면 일괄 변경 → 해당 기간 내 매입 비용 전체 반영
  • 상단 합계 화면에는 영향 없음 (독립 작동)
  • 비용구분 · 매입구분 각각 개별 변경 가능

3

사업관련성 판단 — 업종별 자동화 검토 결과

업종별 추가 케이스 기각 이유
  • Gemini 조건값 증가 → 분류 오류율 증가
  • 재분석 시 전체 데이터 재처리 필요 (100만 건+ 심각)
  • 사업관련성은 주관적 판단 영역 — 자동화 부적합
  • 쿠팡 등 플랫폼 매입 품목 식별 불가
결정: 사업무관 드롭다운 최상단 배치. 신고 전 안내 문구로 주의 환기.
1

셀프텍스 + 원택스 통합 검토

시스템 구조 차이
  • 셀프텍스: 카카오 간편 로그인, 실시간 처리
  • 원택스: 세무대리인 인증서 기반, 자체 배치
  • DB 구조 및 인증 방식이 근본적으로 상이 → 단순 병합 불가
  • 마이그레이션 + 통합 예상 소요: 3~4주
결정: 소득세 완성 이후 후순위. 원택스는 내부 "텍스오피스"로 유지. 신규 기능은 셀프텍스 표준 개발.
2

세무사 B2B 수요 확인

설명회 확인 내용
  • 세무사들이 거래처에 셀프텍스 활용을 강하게 원함
  • 부가세 신고 원가 절감 목적, 설명회 당일 도입 문의 발생
  • 필요 추가 개발: 인증서 분리, 데이터 분리, 로고 커스터마이징
방향: 소득세 우선. B2B는 중장기 로드맵으로 관리.
